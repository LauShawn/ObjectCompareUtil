```powershell
姓名由【张三】更新为【李四】;
门店ID集合由【[1,2]】更新为【[1,3]】;
朋友由【{"Id":2,"Name":null,"ShopIds":null,"Firend":null,"CreateTime":"0001-01-01T00:00:00"}】更新为【{"Id":3,"Name":null,"ShopIds":null,"Firend":null,"CreateTime":"0001-01-01T00:00:00"}】;
```
